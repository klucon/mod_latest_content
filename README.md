# mod_latest_content

Frontend modul pro výpis nejnovějších publikovaných článků z `com_content`.

## Účel

Modul vykreslí krátký seznam posledních publikovaných článků. Hodí se do homepage, sidebaru nebo patičky.

## Poznámky k verzi

První verze s nastavitelným limitem a čistými odkazy `/clanky/<slug>`.
